Clipboard Monitor Documentation

Overview

The Clipboard Monitor is a powerful tool that automatically detects and processes clipboard content. It supports multiple formats including Markdown and Mermaid diagrams.

Features

Core Functionality

• Real-time monitoring of clipboard changes

• Automatic detection of content types

• Seamless conversion between formats

Supported Formats

1. Markdown - Converts to rich text format (RTF)

2. Mermaid - Opens diagrams in browser

3. Plain text - Passes through unchanged

Installation

Prerequisites

Before installing, ensure you have: - Python 3.7 or higher - Required dependencies (see requirements.txt)

Quick Start

# Clone the repository
git clone https://github.com/user/clipboard-monitor.git

# Install dependencies
pip install -r requirements.txt

# Run the monitor
python main.py

Configuration

The application can be configured through the CONFIG dictionary:

CONFIG = {
    'notification_title': 'Clipboard Monitor',
    'polling_interval': 1.0,
    'module_validation_timeout': 5.0
}

Troubleshooting

Common Issues

• Module not loading: Check file permissions

• Conversion failed: Ensure pandoc is installed

• No notifications: Verify macOS notification settings

Error Messages

Error

Solution

pyobjc not found

Install with pip install pyobjc-framework-Cocoa

pandoc not found

Install with brew install pandoc
